# FruitShoot Frontend

Mobile app for the FruitShoot system built with Expo and React Native.

This is currently the frontend for the FruitShoot Application. 

---

## Stack

- Expo
- React Native
- JavaScript
- Expo Router

---

## Requirements

- Node.js 18+
- npm
- Expo Go (iOS or Android)

---

## Install

```bash
npm install
```
## Run

```bash
npx expo start
```

Open with:
- iOS simulator
- Android emulator
- Expo Go (QR code)

---

## Notes
- Phone and computer must be on same network
